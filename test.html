In this example: We have a
element with the class .container, which is styled to act as a container. Inside the container, we have a element with the class .content. We apply the transform property to the .content class to rotate it by 45 degrees. The justify-content and align-items properties are used to center the content both horizontally and vertically within the container. You can adjust the degree of rotation (rotate(45deg)) to your desired angle.
